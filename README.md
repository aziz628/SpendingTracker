# Spending Tracker

Native Android application built for personal expense tracking and financial management.

##  Key Features

*   **Transaction Management**: Full CRUD operations for income and expenses.
*   **Category Systems**: Customizable category management with unique identifiers and icons.
*   **Data Visualization**: Integrated charts and graphs for spending patterns and balance analysis.
*   **User Management**: Support for multiple user accounts with secure local authentication.
*   **Customization**: Personalized experience with dynamic Language and Theme switching.

##  Technical Stack

*   **Platform**: Android (Min SDK 24+)
*   **Language**: Java
*   **Database**: SQLite (Local Persistence)
*   **Build System**: Gradle (Kotlin DSL)
*   **Architecture**: Service-DAO pattern with atomic transaction handling.

## Project Structure

*   `app/src/main/java`: Core logic, models, DAOs, and UI activities.
*   `app/src/main/res/layout`: XML UI designs utilizing a consolidated style hierarchy.
*   `app/src/main/res/values`: Design tokens, themes, and localized resources.

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Project with Gradle Files.
4. Run on an emulator or physical device.
